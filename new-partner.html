Review & Generate

Check the preview, then download.

Steps:
1. Click "Create in HubSpot" to add the partner as a Company + Deal
2. Download the HTML file
3. Save to website-src/partners/[slug].html
4. Add partner logo to website-src/assets/partners/
5. Run: node build.js
6. Deploy: npx wrangler pages deploy website-dist --project-name ascend-site --branch main

Existing partner? Type the name in Step 1 — if it's already in HubSpot, select it to pre-fill the form.
