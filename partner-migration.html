Risks & Open Questions

Logo assets not in repo

The assets/partners/ directory has a .gitkeep but no actual logos. Partner pages show broken images until logos are added. Fallback: /assets/logos/partner-id.webp.

Website deploy is manual

Changes require node build.js then npm run deploy. Commit to main does NOT auto-deploy to Cloudflare Pages.

No per-partner analytics dashboard

UTM params provide channel-level attribution. HubSpot can filter by partner_id contact property if a dashboard is needed.
