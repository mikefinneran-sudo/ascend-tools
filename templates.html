Create New Partner Page

Start typing a partner name to search HubSpot. If the company exists, select it to pre-fill the form. Otherwise, fill in the details to create a new partner.

How Partner Pages Work

Shared form script: All partner pages load onboard.js — handles validation, submission, qualification, and post-submit channel selection (WhatsApp / Voice / Calendly).

Data flow: Form submit dual-posts to Railway webhook + HubSpot Forms API. Partner ID flows through to HubSpot contact properties and WhatsApp bot. UTMs persist via sessionStorage.

To deploy: Save HTML to website-src/partners/[slug].html, add logo to assets/partners/, run node build.js, deploy with wrangler.

Qualification: 2+ trips/mo or $1,000+ avg spend = qualified. Routes to WhatsApp, Voice, or Calendly. Unqualified = waitlist (no follow-up).
